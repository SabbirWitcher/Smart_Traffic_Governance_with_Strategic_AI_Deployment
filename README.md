# Smart_Traffic_Governance_with_Strategic_AI_Deployment

Requirements: Install Python 3.9 or above first.


How to Install:

1. Download the Zip file.
2. Keep the smart_monitoring_system.html and the traffic.mp4 file in the same location.
3. Open CMD in that location by typing CMD in the explorer or by right clicking and selecting open terminal here.
4. In the CMD type this:
  python -m http.server
5. Then in your browser paste this address:
  http://localhost:8000
These two commands are also given in the commands.txt file.
The video tutorial to run this is shwon in the YouTube Video and in the Smart_Monitoring_System.mp4 file.
